ClientHello & FREAK again

Description

The most anticipated OpenSSL announcement finally reveal not less than 14 vulnerabilities, with 2 of them classified as high severity. But don't be fool, that is not an Heartbleed 2 (but patch anyway!).

First, FREAK (CVE-2015-0204) has been reclassified to high because EXPORT_RSA seems to be much more common that previously thought, leading the OpenSSL developpers to escalate it from low to high.

The second high vulnerability (CVE-2015-0291, "ClientHello") only concern the last OpenSSL version (1.0.2), and can lead to a DoS against your server. You can read the full report on the OpenSSL website.

Ubuntu/Debian

aptitude update; aptitude install openssl

Archlinux

pacman -Syu

Red Hat/CentOS

sudo yum update openssl

FreeBSD

freebsd-update fetch; freebsd-update install

Mac OS X

You will have to wait for Apple to wake-up.

Note that patching is currently an ongoing process, so the mirror you use for update may not be in sync with the latest patch yet. If you do not see any openssl package getting upgraded, try a little later.
